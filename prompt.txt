HUNTR AI - COMPLETE TRADING SIGNAL PLATFORM DEVELOPMENT PROMPT (NODE.JS ONLY)
=================================================================================

Build a comprehensive AI-powered trading signal platform called "Huntr AI" that analyzes trading chart images and provides real-time trading signals using Node.js exclusively with the following specifications:

## CORE REQUIREMENTS

**Platform Name**: Huntr AI
**Architecture**: Full-stack Node.js web application with server-side rendering
**Deployment Target**: Vercel, Heroku, or any Node.js hosting platform
**AI Integration**: Gemini 2.5 Pro API for advanced chart image analysis
**Enhancement**: Real-time web search via Serper.dev API for market context

## TECHNICAL STACK (NODE.JS ONLY)

### Backend & Frontend (Node.js/Express)
- Express.js server handling both API routes and web page rendering
- Server-side rendering using template engine (EJS, Handlebars, or Pug)
- MongoDB dual-database architecture (separate user data and training data)
- Express sessions with connect-mongo for persistent authentication
- Image processing pipeline using Sharp and Multer
- Rate limiting and security middleware (helmet, cors, express-session)
- File upload handling with comprehensive validation
- Real-time updates using WebSockets (socket.io) or Server-Sent Events
- Responsive CSS framework (Bootstrap, Tailwind CSS, or custom CSS)
- Vanilla JavaScript for frontend interactivity and AJAX requests
- Static file serving for CSS, JS, and image assets

### Database Design
- **Dual MongoDB Setup**: 
  - Primary database for user data and analysis results
  - Separate training database for AI model improvement
  - Privacy controls allowing users to opt-out of training data collection
- Collections: Users, Analysis, TrainingData
- Proper indexing and relationship management
- Session storage in MongoDB for scalability
